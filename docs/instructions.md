# Hướng dẫn chỉnh sửa nội dung các file

## Đối với các file HTML

> - Sửa ảnh: tìm đến thẻ \<img>, sửa link trong `src`
> - Sửa nội dung: Sửa các thẻ `<a>, <p>, <h123456>, ...`, miễn là sau các thẻ đó và thuộc tính thì có nội dung chữ
> - **Modal** ở điểm du lịch: `Thư mục scripts, file modal.js`
> - **CSS** (_không nên sửa nhiều_): **`CSS/styles.css`**
> - **Ảnh**: Lấy link ảnh _gốc_ hoặc \*tải về rồi up lên thư mục **images\***
